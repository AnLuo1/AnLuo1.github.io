    <h1>Data Challenge Program @ IEEE ICHI 2026 - AgentDS Healthcare</h1>
    <p>
      Human-AI collaboration in healthcare data science.
    </p>
  </section>

  <section class="dc-card">
    <h3>Overview</h3>
    <p>
      The Data Challenge Program at IEEE ICHI 2026 focuses on advancing human-AI collaboration in healthcare data science.
      This program showcases results from the AgentDS Healthcare benchmark, where participants developed reproducible solutions
      for realistic healthcare prediction tasks using collaborative human-AI workflows.
    </p>
    <p>The session brings together researchers and practitioners to:</p>
    <ul class="dc-list">
      <li>Present accepted papers from the Data Challenge track</li>
      <li>Share insights into human-AI collaboration in real-world healthcare settings</li>
      <li>Discuss design principles for reliable, reproducible, and high-impact healthcare data science systems</li>
    </ul>
  </section>

  <div class="dc-grid">
    <section class="dc-card">
      <h3>Theme</h3>
      <p><strong>Human-AI Collaboration in Healthcare Data Science</strong></p>
      <p>The program highlights:</p>
      <ul class="dc-list">
        <li>Realistic healthcare prediction tasks using AgentDS datasets</li>
        <li>Reproducible pipelines and transparent evaluation</li>
        <li>Practical insights into human-AI collaboration workflows</li>
        <li>Lessons for designing next-generation AI-assisted data science systems</li>
      </ul>
    </section>

    <h3>About AgentDS Healthcare</h3>
    <p>
      The program is based on the AgentDS Healthcare benchmark, which provides:
    </p>
    <ul class="dc-list">
      <li>Realistic, domain-specific healthcare datasets</li>
      <li>Multimodal and structured prediction tasks</li>
      <li>Standardized evaluation protocols</li>
